# Beyond the Root | 当我们在讨论主动脉根部时，我们到底在说什么？

![](IMAGE)

问题首先出现在语言之中

当我们在讨论主动脉根部时，问题往往并不首先出现在解剖台上，而是出现在语言之中。这个结构被反复测量、频繁操作，被视为心脏外科中“最基础、也最熟悉”的区域之一。但越是被反复使用，分歧反而越多。

不同学科、不同技术路径、不同训练背景的人，在使用同一套术语时，实际上指向的并不是同一个空间对象。理解的分裂，并非偶然，而是一种长期积累的系统性结果。

技术的进步掩盖了这一问题。影像分辨率不断提高，三维重建越来越精细，器械也愈发可靠，于是人们倾向于相信，对主动脉根部的理解必然随之同步深化。但事实并不总是如此。恰恰是在技术高度成熟的背景下，解剖语言的分裂变得更加隐蔽，也更加危险。

Robert H. Anderson 所指出的“语义危机”，并不是指某一个定义错误，而是多个定义在未被澄清的情况下被同时使用，并且都被默认视为正确。

“瓣环”的双重定义与空间错位

“瓣环”是这一问题最集中的体现。

在外科视角中，它往往被理解为瓣叶的半月形附着线，一个真实存在、可以在术中直接看到并缝合的结构；而在影像学体系中，它更多指向由三个最低附着点在空间中抽象出的虚拟基底平面。

这两种理解在各自语境中都成立，但它们并不等价。一个是解剖实体，一个是几何构造；一个具有组织学厚度和力学属性，另一个只是计算结果。当术前评估依赖后者，而术中操作却基于前者时，所谓“精准评估”其实已经失去了统一对象。

![](IMAGE)

▲“瓣环”并非单一结构：虚拟基底环与真实解剖边界的空间分离

离体标本显示，虚拟基底环（virtual basal ring）、心肌-动脉交界以及瓣叶半月形附着线在三维空间中并不重合。“瓣环”这一术语在不同体系中指向不同空间层级，是主动脉根部解剖语言长期分裂的核心根源。

更深层的问题在于，我们长期习惯于用二维语言描述一个本质上是三维的结构。主动脉瓣叶的附着从来不处在同一水平面上，而是在主动脉壁上形成连续起伏，最低点界定虚拟基底平面，最高点止于交界区水平，整体呈现典型的冠状排列。

这一构型并非解剖细节，而是决定瓣叶力学行为的基础。如果在认知上将其压缩为一个平整的“环”，后续关于瓣叶高度、对合长度和窦部空间的判断，都会在起点就发生系统性偏移。

![](IMAGE)

▲主动脉瓣叶附着线的冠状构型与对合区的空间依赖关系

瓣叶附着线与交界高度在空间中形成明显起伏，对合区并非围绕平面展开，而是受三维几何约束。将该结构误解为二维环形，将直接影响瓣叶高度评估与术中重建决策。

当错误模型进入手术

在瓣膜修复，尤其是瓣膜保留的主动脉根部重建中，这种偏移会被迅速放大。外科医生并不是在“复位”瓣膜，而是在人工构建的空间中重新分配张力、支撑和对合关系。

如果参考模型本身是错误的，即便操作再精细，也只能得到不稳定的结果。许多所谓的“技术失败”，并非源于技巧不足，而是源于对空间现实的误读。

这种误读在术式演进中表现得尤为清晰。以 Modine 提出的简化重植术为例，其吸引力几乎无需解释：步骤更少，时间更短，学习曲线更平缓。在强调效率和可复制性的外科体系中，这样的方案天然具有传播优势。

但简化并非中性操作。它意味着某些结构被视为“可以忽略”，某些约束被判断为“不是必需”。当重植过程仅关注交界顶端的悬吊，而忽略交界间纤维三角区域的结构性重建时，人工血管后方实际上形成了一个缺乏真实支撑的力学空腔。这一问题在短期内可能并不明显，但在反复舒张应力作用下，会逐渐显现其代价。

De Paulis 对重植术的修正，正是对这一现实的直接回应。他们并未试图通过增加复杂性来“战胜”简化，而是重新确认了一个被忽略的事实：交界间纤维三角并非解剖空白，而是力学传递的关键节点。通过在这些区域设置纵向缝合，人工血管被迫顺应真实的三维结构，而非被塑造成简化模型。

这一差异看似细微，却标志着两种根本不同的立场。一种试图通过简化控制复杂性，另一种则选择承认复杂性，并在其基础上重建秩序。

![](IMAGE)

▲交界间纤维三角及其在主动脉根部力学中的关键作用

上图示交界间纤维三角、膜部间隔及主动脉-二尖瓣连续性的空间关系。这些区域并非结构“空隙”，而是瓣叶应力向心室结构传递的核心通道。忽略该区域的重建，将在人工血管后方形成缺乏真实支撑的力学空腔。

从这个角度看，Modine 与 De Paulis 之间的差异，并不仅仅是两种手术技巧的分歧，而是对“主动脉根部究竟是什么”这一根本问题的不同回答。

组织学边界与测量偏差

类似的逻辑同样体现在对组织学边界的理解上。主动脉根部并非由单一组织构成，不同窦部在基底区域所包含的成分存在显著差异。有的区域包含心室肌组织，有的则主要由纤维结构构成。

这种异质性直接决定了缝合线的安全边界，也解释了为何某些并发症呈现出高度选择性的分布。如果在讨论瓣周漏或传导阻滞时，仍然模糊解剖学交界与功能性交界的区别，就无法真正理解问题发生的路径。传导系统并非随机受损，而是位于这些组织交错、语言最容易失焦的区域。

影像评估本应弥合这一鸿沟，但前提是测量本身不制造新的偏差。偏离瓣叶中轴线的切面会系统性低估瓣叶高度，扭曲虚拟基底平面的几何形态。在以毫米为单位进行调整的手术中，这种误差并不微小，它只是被错误地当作“个体差异”所掩盖。工具并不会自动带来真相，除非其背后的模型是清晰的。

当技术走得更远，认知必须对齐

经导管主动脉瓣置入术将这一问题推向极端。在 TAVI 场景中，术者几乎完全依赖影像与虚拟结构进行判断，任何解剖语言或空间模型的不统一，都会被直接转化为并发症风险。

当“瓣环”在不同体系中指向不同对象时，所谓精准植入其实缺乏共同坐标。基础解剖学对传导系统走行的重新澄清之所以能够降低并发症，并不是因为技术突然突破，而是因为认知终于被对齐。

归根结底，主动脉根部的问题并不在于技术是否足够先进，而在于我们是否愿意承认一个并不令人舒适的事实：如果语言、空间模型与测量逻辑无法统一，再精细的操作也只能建立在不稳定的基础之上。

真正的进步并非来自工具的升级，而来自对解剖现实的反复校准。忽视这一点，技术本身反而可能成为放大风险的机制。
