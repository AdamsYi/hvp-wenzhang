# 当患者拿起手术刀：PEARS技术——主动脉医学的“原生保留”革命

在现代心脏外科的殿堂里，绝大多数经典术式都以伟大的外科医生命名：Bentall、David、Yacoub、Ross。这些名字代表了医学界的最高权威，也代表了一种自上而下的“切除与重建”逻辑。

然而，有一项技术的诞生轨迹却截然不同。其发明者既不是医生，也不是医学博士，而是一位马凡综合征（Marfan Syndrome）患者。他的名字叫 Tal Golesworthy。

这项技术被称为 PEARS（个性化外部主动脉根部包裹术）。它的诞生，源于一位顽固的工程师对传统主动脉医学“金标准”发起的根本性质疑。这不是一场简单的医患对话，而是一次工程学思维对医学教条的“降维打击”。

工程师的愤怒：为什么要切掉一个完好的阀门？

2000年，30岁的特许工程师 Tal Golesworthy 收到了一份“判决书”：他的主动脉根部扩张到了危险的临界值。

摆在他面前的只有一条路：Bentall 手术。

这是当时（至今也是）主动脉根部瘤治疗的绝对“金标准”：切除病变的主动脉血管，同时切除主动脉瓣，换上一根带机械瓣的人造血管。手术意味着终身服用华法林抗凝、忍受机械瓣的噪音，以及承担体外循环带来的卒中风险。

作为一名专注于流程工程（Process Engineering）和锅炉管道研究的专家，Tal 感到难以置信。在他眼中，传统主动脉外科的思维显得过于“粗暴”和“浪费”。

他提出了一个纯粹的流体力学质疑：“我的主动脉瓣（阀门）功能是完好的，只是管道壁变薄鼓起来了。为什么要因为管子的问题，把完好的阀门切掉？”

在工业界，当高压管道出现膨胀风险时，工程师绝不会关停整个工厂去更换管道，而是会在管道外部加装一个固定夹（Clamp）或支架，限制其继续膨胀。

他把这个想法告诉了医生：“既然水管鼓包了，为什么不给它外面加个‘箍’，把它勒住不就行了？”

寻找同盟：当 CAD/CAM 遇上心脏外科

当时的医学界对此反应冷淡。医生们认为心脏是动态搏动的复杂器官，简单的外部包裹会导致移位（Migration）或血管壁侵蚀（Erosion），历史上的类似尝试都失败了。

Tal 没有妥协。他利用自己的工程学知识，找到了伦敦皇家布朗普顿医院（Royal Brompton Hospital）的 Tom Treasure 教授和 John Pepper 教授。幸运的是，这两位顶级专家拥有足够的谦卑，愿意倾听一位“外行”的建议。

Tal 开启了一场长达四年的研发。他将工业制造中的 CAD/CAM（计算机辅助设计/制造） 流程首次引入了心脏外科：

数字化重建：通过高分辨率 MRI 扫描，获取患者主动脉根部的 1:1 三维解剖数据。

3D 打印模具：打印出患者那个扩张的主动脉实物模型。

定制“外骨骼”：在模型上利用医用高分子材料（ExoVasc®），编织一个与患者血管形状毫厘不差的网套。

![](IMAGE)

图 1：Tal Golesworthy 手持基于他本人心脏数据 3D 打印的主动脉根部模型，向医学界解释他的工程学构想。

这个网套的设计充满了工程学的智慧：它拥有 0.7mm 的微孔结构。这个孔径既保证了机械强度，又能让患者的外膜细胞穿过网孔生长，最终使网套与血管壁融为一体，彻底解决了“移位”的难题。

图 2：ExoVasc® 个性化网套。这个看似简单的织物，是全球第一个通过 CAD/CAM 流程为心脏完全定制的植入物。

2004年：第一例“小白鼠”

2004年5月24日，Tal Golesworthy 躺上了手术台。他是这项技术的发明者，也是世界上第一位接受 PEARS 手术的患者。

手术过程验证了“极简主义”的胜利。外科医生不需要切开主动脉，不需要切除瓣膜，甚至不需要让心脏停跳。他们只需将那个定制好的网套，像穿袜子一样套在扩张的主动脉外侧。

![](IMAGE)

图 3：PEARS 手术原理。网套包裹在主动脉外侧（External Support），物理限制了直径扩张，从而降低了室壁张力。

当网套就位，主动脉的直径被物理限制在了安全范围内。根据 Laplace 定律（室壁张力 = 压力 × 半径），随着半径被限制，作用在脆弱血管壁上的张力瞬间降低，夹层风险随之解除。

最重要的是，这一过程完整保留了患者的自体组织。没有任何东西被切除，没有任何异物（如机械瓣）接触血液。

从“异端”到“选项”

Tal Golesworthy 赌赢了。

术后20年，他依然健康地活着。影像学随访显示，他的主动脉根部尺寸极其稳定，自体瓣膜功能完好，且无需服用任何抗凝药物。他像正常人一样生活，甚至继续从事他的工程咨询工作。

随着首例的成功，PEARS 逐渐从“异端”走向临床。它不再是单一患者的自救，而成为了一种标准化的医疗解决方案。它为那些年轻的马凡综合征患者提供了一个珍贵的第三选项：不需要在“机械瓣抗凝”和“生物瓣衰败”之间做两难选择，而是保留自己的器官，给予它一个坚强的后盾。

尾声：医学的谦卑

PEARS 技术的诞生，提醒了整个主动脉医学界：面对复杂的解剖病变，“切除与置换”（Resection and Replacement）未必是唯一的答案。

通过精准的“外部支持”（External Support），我们完全可以在保留生理功能的前提下，遏制病程的进展。Tal Golesworthy 用自己的生命证明了：有时候，最好的医生，是那个最了解自己身体痛苦的患者；而最好的手术刀，可能是电脑里生成的 3D 模型。
