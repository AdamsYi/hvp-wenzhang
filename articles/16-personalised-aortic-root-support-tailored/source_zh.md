# 个体化主动脉根部支撑：马凡综合征患者的「量体裁衣」之选

A tailored solution preserves the architecture of the native valve.

对于马凡综合征（Marfan syndrome）患者而言，主动脉根部动脉瘤及随之而来的主动脉夹层，往往是悬在生命线上最沉重的阴影。为了防患于未然，传统的标准干预策略通常是进行全主动脉根部置换（total root replacement, TRR）或保瓣主动脉根部置换（valve-sparing root replacement, VSRR）。然而，这两种手术不仅技术要求高，且往往伴随着终身抗凝或较高的再干预风险。2004年，一种名为个体化外部主动脉根部支撑（personalised external aortic root support, PEARS）的创新保守疗法应运而生。《Heart》杂志发表了一项前瞻性研究，详细报告了前30名连续接受PEARS手术的马凡综合征患者长达1至9年的随访结局，为这一技术的可行性与远期疗效提供了关键证据。

这项研究由伦敦皇家布朗普顿医院和比利时鲁汶大学联合开展，共纳入了在2004年至2011年间接受PEARS手术的30例患者。这批患者的中位年龄为28岁（IQR 20-44），术前主动脉根部平均直径为46.2毫米（SD 3.4）。PEARS技术的核心在于「量体裁衣」：研究团队首先利用患者的MRI或CT空间数据进行计算机辅助设计（CAD），通过3D打印技术制作出与患者主动脉完全一致的物理模型（former）。随后，在这个模型上编织出一件由医疗级聚合物纱线制成的、具有大孔径的柔软网状「外套」。在手术中，医生在心脏跳动的情况下，将这件定制的网状支撑物紧密包裹在患者的主动脉根部，并延伸至头臂动脉之外——这一特性使其比传统根部置换手术的保护范围更广。

最令人瞩目的数据来自于术后的长期随访。在总计133个患者-年的随访期内（平均随访4.4年，最长达8.8年），这30名患者中没有发生任何死亡事件，也没有出现脑血管事件、主动脉或瓣膜相关的并发症。在随访结束时，有29名患者的心功能处于NYHA I级，能够正常工作或学习。唯一一名NYHA III级患者是由于术前即存在的其他严重合并症（如慢性阻塞性肺病等）所致，与主动脉根部疾病无关。简单来讲就是，这件量身定制的「外套」不仅成功阻止了主动脉的进一步扩张，还让绝大多数患者恢复了高质量的正常生活，而无需承担传统置换手术带来的沉重负担。

与金标准的比较

当我们将目光转向传统手术标准时，PEARS的优势显得尤为独特。研究团队将这30例PEARS患者的结局，与已发表的包含1385例接受TRR或VSRR手术患者的荟萃分析数据进行了对比。数据显示，TRR的早期死亡率为4.1%（95% CI 1.9-7.7），血栓栓塞率为0.7%/年（0.5-0.9）；VSRR的早期死亡率为3.2%（0.5-17.9），瓣膜再干预率高达1.3%/年（0.3-2.2）。相比之下，PEARS队列在这些核心并发症上均保持了「零」的记录。此外，PEARS手术的创伤显著更小——30例患者中仅有1例使用了极短时间（20分钟）的体外循环，中位失血量仅为150毫升（IQR 75-469），中位总住院时间为8天（IQR 6.0-9.3）。

这种对比的价值固然值得重视，但也需要审慎解读。接受PEARS手术的患者在基线上通常比接受TRR或VSRR的患者更年轻（平均31岁 vs 35岁/33岁）、主动脉直径更小（46.2 mm vs 61 mm/52 mm），且没有夹层病史。这种基线差异使得直接的比较不可避免地带有一定的提前期偏倚（lead time bias）——患者在更小的主动脉直径时接受手术，本身就意味着在随访期内面临更低的自然夹层风险。文献作者对此坦诚相告，并指出只有通过前瞻性随机对照研究，才能真正厘清这种差异。

「外骨骼」的生物学逻辑

面对这样一项颠覆传统的新技术，学术界自然也提出了诸多审慎的疑问。最集中的顾虑包括：支撑物是否会移位并压迫冠状动脉？主动脉壁是否会因包裹而变薄？支撑范围内是否仍会发生夹层？对此，研究者通过长期的动物实验和临床观察给出了回应。大孔径的网状结构能够让主动脉外层的细胞长入其中，实现生物性融合，从而彻底杜绝了移位的可能；同时，羊颈动脉模型证实，被包裹的血管壁实际上会增厚而非变薄，内皮层始终保持完整。通俗点说，这层网并非只是简单地「套」在血管外面，而是随着时间的推移，真正长成了血管壁的一部分，成为了坚固的「外骨骼」。

在整个随访期间，没有任何一例患者出现支撑物相关的冠状动脉受压事件，也没有发现主动脉壁变薄的影像学证据。7年累积生存率达到100%。这些数据在一定程度上回应了外界的顾虑，但也需要指出，这毕竟是一个样本量较小（n=30）、随访时间相对有限的初步研究，更长期的随访数据和更大规模的比较研究仍是必要的。为了获得更确凿的证据，一项名为BARS（Big Aortic Root Study）的大型前瞻性比较研究正在筹备中。

对于那些在年轻时就被诊断为马凡综合征、面临着主动脉随时可能撕裂风险的患者来说，PEARS技术的出现无疑提供了一种充满希望的新选择。它最大程度地保留了患者原有的解剖结构和生理功能，避免了人工瓣膜带来的长期抗凝烦恼，也规避了体外循环的巨大创伤。这项来自皇家布朗普顿医院的早期数据，客观呈现了一个在133个患者-年随访中未见任何主动脉或瓣膜相关事件的初步结果，其所代表的技术路径值得在更严格的前瞻性框架下继续探索。
